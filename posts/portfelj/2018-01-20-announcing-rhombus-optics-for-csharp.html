Rhombus - Optics for C# released

January 20, 2018
A small high-level library to model relationships between types

We are proud to announce the availability of Rhombus, an optics library for C# and dotnet, which we have ported from the popular F# library Aether by Xyncro Ltd.

Check the GitHub repo for more information.

Also available as a NuGet package!
